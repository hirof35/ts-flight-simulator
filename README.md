# 2D TypeScript 航空力学シミュレーター

TypeScriptとHTML5 Canvasで作成した、2D真横視点の簡易フライトシミュレーターです。
<img width="1000" height="779" alt="スクリーンショット 2026-06-01 084041" src="https://github.com/user-attachments/assets/b5c840c2-f405-4818-8b05-f340c3c67aa6" />

## 特徴
- **航空力学の再現**: 揚力、抗力、推力、重力の4つの力を毎フレーム計算しています。
- **失速（ストール）モデル**: 迎角（AoA）が一定を超えると、主翼の揚力が激減して墜落する挙動をリアルに再現しています。
- **滑らかな操作性**: マウス（ピッチ操作）とキーボード（無段階スロットル）による快適な操縦が可能です。

## 操作方法
- **機首上げ・下げ**: マウスの上下移動
- **加速（スロットルON）**: キーボード `→` キーを押し続ける
- **減速（スロットルOFF）**: キーボード `←` キーを押し続ける
